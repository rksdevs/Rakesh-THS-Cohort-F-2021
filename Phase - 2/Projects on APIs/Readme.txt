This folder has various projects which I've done based on available APIs. 

APIs used: 
    1. Battery API
    2. Weather API - openweathermap.org
    3. Hackernews API - hacker-news.org

Node Modules used: 
    1. Axios - To fetch data from API
    2. File System - Node's inbuilt FS module to read, write and append data into files
    3. Readline-sync - To accept inputs from command line

Features:
    1. These are simple JS applications
    2. These projects showcase the use of APIs to fetch data
    3. These projects showcase how to handle asynchronous JS
    4. These projects showcase how to use FS module of node js
